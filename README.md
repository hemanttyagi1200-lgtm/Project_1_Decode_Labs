## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/hemanttyagi1200-lgtm/Project_1_Decode_Labs.git
```

Go into the project folder:

```bash
cd Project_1_Decode_Labs
```

Run the chatbot:

```bash
python rule_based_chatbot.py
```
